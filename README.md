# Step8up AI Essentials Quiz

An interactive AI knowledge quiz built for the Step8up expo. Designed for iPad touch screens with a techy dark theme.

## Features

- 8 multiple-choice questions based on the BCS AI Essentials syllabus
- Touch and click friendly — works on iPad and desktop
- Auto-advances to the next question after 2 seconds
- Animated code typewriter effect behind quiz questions
- Percentage score with personalised feedback
- Hosted on GitHub Pages — no server required

## Tech Stack

- **React 18** (CDN)
- **Tailwind CSS v3** (CDN)
- **Babel Standalone** for in-browser JSX
- Single `index.html` — no build step needed

## Run Locally

```bash
cd "AI quiz"
python -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080)
